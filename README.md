#### MODULE NAME
       INC4

#### DEPENDENCIES
       MUX4

#### INPUTS
Name | Size(Bits) 
-----|------------
A[3] |     1      
A[2] |     1      
A[1] |     1      
A[0] |     1      

#### OUTPUTS
Name | Size(Bits)
-----|------------
S[3] |     1      
S[2] |     1      
S[1] |     1      
S[0] |     1      

#### DESCRIPTION
       Takes a 4-bit input A (bits are wired into separate inputs), and gener-
       ates a four-bit output S = A + 1 (bits are  wired  into  separate  out-
       puts).
